### Проверка целостности кабеля

```txt
Пожалуйста, проверьте основной интернет-кабель, выходящий из подъезда
Есть ли на нём видимые повреждения или пережатые участки?
```

  <starlight.TabItem label="Напрямую">
    ```txt
    Давайте проверим получает ли компьютер сигнал:
    1. На вашем ПК откройте Пуск (значок Windows в левом углу)
    2. В поиске введите Панель управления и откройте её
    3. Нажмите на Сеть и Интернет, далее Центр управления сетями и общим доступом
    4. В этом окне нажмите слева на Изменения параметров адаптера и нажмите правой кнопкой мыши на Подключение по
    локальной сети (если такого нет, подключение может называться Ethernet)
    5. Далее откройте Состояние

    Что указано в "Состояние среды"? Отправьте, пожалуйста, фото
    ```
  </starlight.TabItem>

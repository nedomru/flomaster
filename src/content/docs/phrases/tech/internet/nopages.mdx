### Установка статических метрик

        ```txt "[IP КЛИЕНТА]" "[ШЛЮЗ КЛИЕНТА]" "[DNS1 КЛИЕНТА]" "[DNS2 КЛИЕНТА]"
        1. Откройте настройки Wi-Fi на телефоне
        2. Нажмите на шестеренку рядом с вашей сетью
        3. Вверху справа нажмите на карандаш - "Расширенные настройки" - "Настройки IP-протокола"
        4. Укажите параметры:
        - Статический IP-адрес

        - Шлюз - [ШЛЮЗ КЛИЕНТА]

        - DNS 2 - [DNS2 КЛИЕНТА]

        ```

### Проверка активности прокси
<Tabs>
    <TabItem label="Win 10" icon="seti:windows">
        ```
        Для проверки активности прокси:
        1. Откройте "Пуск" - "Параметры" - "Сеть и Интернет"
        2. В левой панели перейдите в раздел "Прокси-сервер"
        3. Пролистайте вниз до раздела "Настройка прокси вручную"

        Уточните, активна ли галочка на пункте "Использовать прокси-сервер"?
        ```
    </TabItem>
    <TabItem label="Win 11" icon="seti:windows">
        ```
        Для проверки активности прокси:
        1. Откройте "Пуск" - "Параметры" - "Сеть и Интернет"
        2. Далее перейдите в раздел "Прокси-сервер" и нажмите кнопку "Настройка"

        Уточните, активна ли галочка на пункте "Использовать прокси-сервер"?
        ```
    </TabItem>
</Tabs>

### Удаление сети Wi-Fi

<Tabs>
    <TabItem label="Android" icon="linux">
        ```txt
        Сейчас нужно удалить сохраненную сеть:
        1. Откройте настройки Wi-Fi на телефоне
        2. Нажмите на шестеренку рядом с вашей сетью
        3. Нажмите кнопку "Удалить"
        ```
    </TabItem>

    <TabItem label="iOS" icon="apple">
        ```txt
        Сейчас нужно удалить сохраненную сеть:
        1. Откройте настройки Wi-Fi на телефоне
        2. В самом верху найдите сеть с галочкой слева. Нажмите на значок "i" справа от нее
        3. В открывшемся окне нажмите "Забыть эту сеть" и подтвердите
        ```
    </TabItem>

    <TabItem label="Win" icon="seti:windows">
        <Tabs>
            <TabItem label="Win10" icon="seti:windows">
                ```txt
                Сейчас нужно удалить сохраненную сеть:
                1. Нажмите на иконку подключения к интернету на панели уведомлений (в правом нижнем углу)
                2. Откройте "Параметры сети" - "Управление параметрами сети Wi-Fi"
                3. Выберите из списка Wi-Fi вашу сеть и нажмите кнопку "Забыть"
                ```
            </TabItem>

            <TabItem label="Win8" icon="seti:windows">
                ```txt
                Сейчас нужно удалить сохраненную сеть:
                1. Откройте список доступных Wi-Fi сетей
                2. Нажмите на вашу сеть правой кнопкой мыши и нажмите "Забыть эту сеть"
                ```
            </TabItem>

            <TabItem label="Win7" icon="seti:windows">
                ```
                Сейчас нужно удалить сохраненную сеть:
                1. Откройте "Пуск" - "Панель управления" - "Сеть и интернет" - "Центр управления сетями и общим
                доступом"
                2. Слева откройте "Управление беспроводными сетями"
                3. Найдите вашу сеть в списке и удалите ее
                ```
            </TabItem>
        </Tabs>
    </TabItem>

    <TabItem label="Apple" icon="apple">
        ```txt
        Сейчас нужно удалить сохраненную сеть:
        1. Откройте "Системные настройки" - "Wi-Fi"
        2. Нажмите "Подробнее" рядом с сетью, которую вы используете
        3. Нажмите "Забыть эту сеть", а затем "Удалить" для подтверждения
        ```
    </TabItem>
</Tabs>

### Очистка кеша браузера

<Tabs>
    <TabItem label="ПК">
        ```txt
        Потребуется очистить кеш браузера, выполните следующие действия:
        1. В открытом браузере нажмите комбинацию клавиш Ctrl + Shift + Delete
        2. В появившемся окне очистки истории отметьте следующие пункты:
        - Файлы cookie (куки)
        - Кеш браузера
        3. В выпадающем меню временного периода выберите "За всё время" или "Весь период"
        4. Нажмите кнопку "ОК" или "Очистить данные"

        Для применения изменений перезапустите браузер
        ```
    </TabItem>

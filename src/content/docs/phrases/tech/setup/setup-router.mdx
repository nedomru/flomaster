### Acorp / Акорп

#### Вход в настройки
```txt
Для входа в настройки роутера:
1. Подключитесь к сети роутера по кабелю или Wi-Fi
2. Откройте сайт http://192.168.1.1
3. На странице войдите с данными от роутера (admin / admin1)
```

#### Настройка соединения
<Tabs>
    <TabItem label="PPPoE">
        ```txt "Логин PPPoE" "Пароль PPPoE"
        Настроим соединение:
        1. Откройте раздел Operation Mode
        2. Напротив значения "AP-Gateway" включите галочку
        3. Откройте раздел "Internet Settings" - "WAN Setup"
        4. Укажите параметры:
        - Wan Connection Type - "DHCP (Auto Config)"
        - Enable NAT - включите галочку
        5. Откройте раздел "Internet Settings" - "VPN Setup"
        6. Укажите параметры:
        - Enable VPN - включите галочку
        - VPN Mode - "PPPoE Client"
        - PPPoE interface - "WAN"
        - Authentication method - "Auto"
        - Username - Логин PPPoE
        - Password - Пароль PPPoE
        - Default gateway - "Enabled"
        - Adaptive LCP, Enable NAT, Peer DNS - включите галочки

        Нажмите кнопку "Apply and connect"
        ```
    </TabItem>
</Tabs>

#### Настройка Wi-Fi
```txt
Настроим Wi-Fi:
1. Откройте раздел "Wireless Settings" - "Basic"
2. Укажите параметры:
- Wireless enable - включите галочку
- Network Mode - "11b/g/n mixed"
- Network Name (SSID) - желаемое название Wi-Fi
- Broadcast Network Name (SSID) - включите галочку
Нажмите кнопку "Apply"

3. Откройте раздел "Security"
4. Установите параметры:
- SSID choice - выберите точку доступа, для которой устанавливаете пароль (ваша сеть)
- Security Mode - "WPA2 PSK"
- WPA Algorithms - "TKIP+AES"
- Pass Phrase - желаемый пароль Wi-Fi
- Policy - выключите галочку
Нажмите кнопку "Apply"
```

#### Смена каналов
```txt
Сменим каналы Wi-Fi:
1. Откройте раздел "Wireless Settings" - "Basic"
2. Укажите параметры:
- Frequency - "2417 (Channel 2)"
- Channel BandWidth - "20/40"

Нажмите кнопку "Apply"
```

#### Настройка Wi-Fi
```txt
Настроим Wi-Fi:
1. Откройте раздел "Беспроводная сеть"
2. Укажите параметры:
- Режим сети - "Создание беспроводной сети"
- Имя беспроводной сети - желаемое название Wi-Fi
- Защита беспроводной сети - "Личный WPA2"
- Беспроводной пароль - желаемый пароль Wi-Fi
- Подтверждение пароля - желаемый пароль повторно
- Запомнить этот пароль в связке ключей - включите галочку

Нажмите кнопку "Обновить"
```

#### Смена каналов
```txt "КАНАЛ 2.4 ГГц" "КАНАЛ 5 ГГц"
Сменим каналы Wi-Fi:
1. Откройте раздел "Беспроводная сеть"
2. Нажмите кнопку "Edit" и установите параметры:
- 2.4 ГГц:
    - Канал - КАНАЛ 2.4 ГГц
    - Ширина канала - "20MHz"
- 5 ГГц:
    - Канал - КАНАЛ 5 ГГц
    - Ширина канала - "Авто (20/40/80)"

Нажмите кнопку "Обновить"
```

#### Проверка MAC-адреса
```txt
Чтобы узнать MAC-адрес вашего роутера:
1. На Mac откройте папку "Программы" - "Утилиты" - "AirPort"
2. В программе выберите ваш роутер
3. Отправьте фото или напишите значение пункта "MAC-адрес"
```

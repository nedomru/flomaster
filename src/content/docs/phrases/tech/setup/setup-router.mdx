### Acorp

#### Вход в настройки
```txt
1. Подключитесь к сети роутера по кабелю или Wi-Fi
2. Откройте сайт http://192.168.1.1
3. На странице войдите с данными от роутера (admin / admin1)
```

#### Настройка соединения
<starlight.Tabs>
    <starlight.TabItem label="PPPoE">
        ```txt
        1. Откройте раздел Operation Mode
        2. Напротив значения "AP-Gateway" включите галочку
        3. Откройте раздел "Internet Settings" - "WAN Setup"
        4. Укажите параметры:
        - Wan Connection Type - "DHCP (Auto Config)"
        - Enable NAT - включите галочку
        5. Откройте раздел "Internet Settings" - "VPN Setup"
        6. Укажите параметры:
        - Enable VPN - включите галочку
        - VPN Mode - "PPPoE Client"
        - PPPoE interface - "WAN"
        - Authentication method - "Auto"
        - Username и Password - введите данные по договору от PPPoE
        - Default gateway - "Enabled"
        - Adaptive LCP, Enable NAT, Peer DNS - включите галочки

        Нажмите кнопку "Apply and connect"
        ```
    </starlight.TabItem>
</starlight.Tabs>

#### Настройка Wi-Fi
```txt
1. Откройте раздел "Wireless Settings" - "Basic"
2. Укажите параметры:
- Wireless enable - включите галочку
- Network Mode - "11b/g/n mixed"
- Network Name (SSID) - желаемое название Wi-Fi
- Broadcast Network Name (SSID) - включите галочку
Нажмите кнопку "Apply"

3. Откройте раздел "Security"
4. Установите параметры:
- SSID choice - выберите точку доступа, для которой устанавливаете пароль (ваша сеть)
- Security Mode - "WPA2 PSK"
- WPA Algorithms - "TKIP+AES"
- Pass Phrase - желаемый пароль Wi-Fi
- Policy - выключите галочку
Нажмите кнопку "Apply"
```

#### Смена каналов
```txt
1. Откройте раздел "Wireless Settings" - "Basic"
2. Укажите параметры:
- Frequency - "2417 (Channel 2)"
- Channel BandWidth - "20/40"

Нажмите кнопку "Apply"
```

#### Настройка Wi-Fi

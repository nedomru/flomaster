### Настройка PPPoE
<Tabs>
    <TabItem label="Win7" icon="seti:windows">
        ```txt
        Для настройки соединения на Windows 7:
        1. Откройте "Пуск" - "Панель управления" - "Сеть и Интернет"
        2. Выберите "Центр управления сетями и общим доступом" - "Настройка нового подключения или сети"
        3. В появившемся окне выберите пункт "Подключение к Интернету"
        4. Нажмите на "Высокоскоростное (с PPPoE)" и введите имя пользователя и пароль по договору от PPPoE
        5. Нажмите "Подключить"
        ```
    </TabItem>
    <TabItem label="Win8" icon="seti:windows">
        ```txt
        Для настройки соединения на Windows 8:
        1. Откройте "Пуск" - "Панель управления" - "Сеть и Интернет"
        2. Выберите "Центр управления сетями и общим доступом" - "Настройка нового подключения или сети"
        3. В появившемся окне выберите пункт "Подключение к Интернету"
        4. Нажмите на "Высокоскоростное (с PPPoE)" и введите имя пользователя и пароль по договору от PPPoE
        5. Нажмите "Подключить"
        ```
    </TabItem>
    <TabItem label="Win8.1" icon="seti:windows">
        ```txt
        Для настройки соединения на Windows 8.1:
        1. Правой кнопкой мыши нажмите по значку сетевых подключений на панели задач и
        выберите пункт "Центр управления сетями и общим доступом"
        2. Выберите "Создание и настройка нового подключения или сети"
        3. Выберите "Подключение к Интернету" и нажмите кнопку "Далее"
        4. Нажмите на "Высокоскоростное (с PPPoE)" и введите имя пользователя и пароль по договору от PPPoE
        5. Нажмите "Подключить"
        ```
    </TabItem>
    <TabItem label="Win10" icon="seti:windows">
        ```txt
        Для настройки соединения на Windows 10:
        1. Откройте "Пуск" - "Параметры" - "Сеть и Интернет" - "Набор номера"
        2. Затем нажмите на "Настройка нового подключения"
        3. Откроется мастер создания сетевых подключений. Выбираем подключение к сети интернет и нажимаем Далее
        4. Нажмите на "Высокоскоростное с РРРоЕ" и введите имя пользователя и пароль по договору от PPPoE
        5. Нажмите "Подключить"
        ```
    </TabItem>
    <TabItem label="Win11" icon="seti:windows">
        ```txt
        Для настройки соединения на Windows 11:
        1. Откройте "Пуск" - "Параметры" - "Сеть и Интернет" - "Набор номера"
        2. Затем нажмите на кнопку "Настройка нового подключения"
        3. Откроется меню создания новых подключений, выберите "Подключение к Интернету"
        4. Тип подключения выберите "Высокоскоростное с РРРоЕ"
        5. Введите имя пользователя и пароль по договору от PPPoE, нажмите "Подключить"
        ```
    </TabItem>
    <TabItem label="Mac" icon="apple">
        ```txt
        Для настройки соединения на Mac:
        1. Откройте "Системные настройки" - "Сеть"
        2. Откройте меню действий справа (в виде трех точек), выберите "Добавить службу"
        3. Оставьте интерфейс Ethernet, имя службы введите на свое усмотрение (Например, dom.ru), далее "Создать"
        4. В списке сетевых служб выберите созданную вами службу, нажмите кнопку "Подробнее"
        5. Введите имя службы и имя пользователя от PPPoE
        6. Включите параметр "Запомнить пароль", затем введите пароль от PPPoE
        ```
    </TabItem>
</Tabs>

### Настройка Static IP
<Tabs>
    <TabItem label="Win" icon="seti:windows">
        ```txt
        Для настройки соединения на Windows:
        1. Нажмите на клавиатуре WIN+R, в окне введите "ncpa.cpl" без кавычек
        2. В открывшемся окне нажмите правой кнопкой мыши на "Подключение по локальной сети" или "Ethernet" - "Свойства"
        3. В списки найдите "Протокол Интернета версии 4" (или "IP версии 4"), дважды нажмите на него
        4. В окне выберите "Использовать следующий IP адрес" и введите данные по договору
        ```
    </TabItem>
    <TabItem label="Mac" icon="apple">
        ```txt
        Для настройки соединения на Mac:
        1. Откройте раздел "Системные настройки" - "Сеть"
        2. Откройте меню действий справа (в виде трех точек), выберите "Добавить службу"
        3. Оставьте интерфейс Ethernet, имя службы введите на свое усмотрение (Например, dom.ru), далее "Создать"
        4. В списке сетевых служб выберите созданную вами службу, нажмите кнопку "Подробнее"
        5. Откройте пункт "TCP/IP", "Конфигурация IP" установите в "Вручную" и введите данные по договору
        6. Перейдите в пункт "DNS", нажмите на кнопку "+" в строке добавления DNS, данные также введите по договору
        ```
    </TabItem>
</Tabs>

### Смена MAC-адреса устройства
<Tabs>
    <TabItem label="Win">
        ```txt "[ВВЕДИ MAC]"
        Требуется сменить MAC-адрес:
        1. Нажмите на клавиатуре WIN+R, в окне введите "devmgmt.msc" без кавычек
        2. В открывшемся окне разверните раздел "Сетевые адаптеры"
        3. Кликните правой кнопкой мыши по вашей сетевой карте и выберите "Свойства"
        4. Откройте вкладку "Дополнительно" - "Сетевой адрес"
        5. В поле "Значение" введите [ВВЕДИ MAC]

        По завершении инструкции нажмите ОК и перезагрузите компьютер
        ```
    </TabItem>
    <TabItem label="Mac" icon="apple">
        ```txt "[ВВЕДИ MAC]"
        Требуется сменить MAC-адрес:
        1. Откройте Терминал на компьютере
        2. В открывшемся окне введите "sudo ifconfig en0 ether [ВВЕДИ MAC]" без кавычек
        3. Нажмите Enter

        После выполнения инструкции покажите на фото или скриншотом вывод в окне
        ```
    </TabItem>
</Tabs>

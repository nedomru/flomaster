### Humax 9000i

#### Проверка NUID

<starlight.Tabs>
    <starlight.TabItem label="Через настройки">
        ```txt
        Нужно уточнить номер приставки:
        1. Нажмите кнопку "Меню" на пульте от приставки
        2. Откройте раздел "Настройки" - "Система" - "Информация" - "Smart Card"

        На открывшейся странице найдите значение пункта "Chipset NUID" (12 цифр)
        Напишите его, пожалуйста, или пришлите его фото в чат
        ```
    </starlight.TabItem>

    <starlight.TabItem label="С наклейки">
        ```txt
        Нужно уточнить номер приставки:
        Снизу на приставке, где наклейка, будет написан её NUID в формате 10 цифр
        Напишите его, пожалуйста, или пришлите фото наклейки в чат
        ```
    </starlight.TabItem>
</starlight.Tabs>

#### Проверка сетевых метрик

```txt
Проверим корректность подключения приставки к сети:
1. Нажмите кнопку Дом.ru на приставке
2. Откройте раздел "Настройки" - "Настройки" (значок шестеренки) - "Настройки интернета" - "Конфигурация LAN"

На открывшейся странице проверьте:
- "Настроить IP" - Автоматически
- "Настроить DNS" - Автоматически

Далее направьте либо адреса, указанные на странице, либо фото страницы в чат
```

#### Проверка версии ПО

```txt
Требуется проверить версию приставки:
1. Нажмите кнопку Дом.ru на приставке
2. Откройте раздел "Настройки" - "Система" - "Информация о системе"

На открывшейся странице найдите значение пунктов "Программная версия", "Версия оболочки" и "Версия загрузчика"
Далее направьте их текстом, либо на фото в чат
```

#### Проверка автоотключения

```txt
Проверим настроенное автоотключение на приставке:
1. Нажмите кнопку Дом.ru на приставке
2. Откройте раздел "Настройки" - "Система"

На открывшейся странице найдите значение пункта "Автоматическое выключение"
Убедитесь, что оно установлено в "выключено"
```

#### Перенастройка приставки
```txt
Потребуется перенастройка приставки:
1. Нажмите кнопку Дом.ru на приставке
2. Откройте раздел "Настройки" - "Настройки" (значок шестеренки) - "Сброс к заводским установкам"
3. Подтвердите сброс, выбрав "Да, сбросить все настройки"

После перезагрузки пройдите по инструкциям на экране ТВ
```

#### Проверка номера карты

```txt
Нужно уточнить номер карты:
Карта находится в декодере, справа под крышкой (на крышке наклейка Дом.ru)
Потяните крышку на себя (справа есть выемка), она откроется, извлеките карту

Сообщите какой номер на ней указан
```

#### Ошибка карты на экране ТВ

```txt
Если на экране сообщение "Ошибка карты", то необходимо проверить саму карту
Карта находиться в декодере, справа под крышкой (на крышке наклейка Дом.ru)

Потяните крышку на себя, она откроется, извлеките карту
Чип на карте можно протереть чистой сухой тканью

Установите карту обратно плотно - чипом вниз, от себя
После этого проверьте работу каналов, которые не показывали
```

#### Перенастройка приставки
```txt
Потребуется перенастройка приставки:
1. Нажмите кнопку "Меню" на пульте от приставки
2. Откройте раздел "Настройки" - "Заводские настройки"
3. Введите пароль для подтверждения сброса - 0000

После перезагрузки пройдите по инструкциям на экране ТВ
```

            ```txt
            3. Зажмите кнопку питания и держите ее

            Пульт начнет подбор кода и передачу команды отключения телевизора
            Когда телевизор выключится - нажмите кнопку "ОК" на пульте для сохранения
            ```

            ```txt
            Для привязки пульта к телевизору:
            1. Зажмите кнопку "SET" на 3 секунды, пока индикатор на пульте не станет ярко-красным
            2. Нажмите на пульте кнопку, на которую нужно назначить функцию. Индикатор начнет мигать
            3. Направьте пульт от телевизора на наш пульт инфракрасным датчиком на расстоянии 3-5см, и нажмите на старом пульте кнопку для программирования нашего пульта
            4. После успешного программирования индикатор промигает 3 раза и начнет гореть непрерывно. Далее можете повторить 3 пункт с другими кнопками

            После настройки нажмите кнопку "SET" для сохранения, красный индикатор должен погаснуть
            ```
            </TabItem>

            ```txt
            Для привязки пульта к телевизору:
            1. Направьте пульт от телевизора на наш пульт инфракрасным датчиком на расстоянии 1см
            2. На нашем пульте зажмите кнопку "Вкл/выкл ТВ" на 5 секунд. Индикатор на пульте должен начать гореть непрерывно
            3. На нашем пульте нажмите кнопку, которую хотите настроить. Индикатор должен начать мигать
            4. На пульте от телевизора нажмите кнопку для программирования нашего пульта
            5. Через 3 секунды индикатор снова начнет гореть непрерывно, пульт будет готов к программированию других кнопок

            После настройки нажмите кнопку "MENU" для сохранения, индикатор должен погаснуть
            ```

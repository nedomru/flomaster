### Проверка MAC-адреса

<Tabs>
    <TabItem label="Через настройки">
        ```txt
        Нужно уточнить номер приставки:
        1. Нажмите на пульте "Меню"
        2. Откройте раздел "Настройки" - "О системе"

        На открывшейся странице найдите значение "физический адрес"
        Напишите его, пожалуйста, или пришлите его фото в чат
        ```
    </TabItem>

    <TabItem label="С наклейки">
        ```txt
        Нужно уточнить номер приставки:
        Снизу на приставке, где наклейка, будет написан её MAC-адрес в формате XX:XX:XX:XX:XX:XX
        Напишите его, пожалуйста, или пришлите фото наклейки в чат
        ```
    </TabItem>
</Tabs>

### Проверка сетевых метрик

```txt
Проверим корректность подключения приставки к сети:
1. Перезагрузите приставку по питанию. Во время загрузки приставки появится информация о IP адресах в левом нижнем углу
2. При появлении этой информации необходимо нажать на любую кнопку на пульте от приставки

После кнопки появится дополнительное окно с сетевыми настройками
Убедитесь, что вверху слева окна указано "DHCP"

Далее напишите в чат указанные в окне адреса, либо направьте их фото в чат
```

### Проверка версии ПО
```txt
Требуется проверить версию приставки:
1. Нажмите на пульте "Меню"
2. Откройте раздел "Настройки" - "О системе"

На открывшейся странице найдите пункт "Версия ПО"
Напишите значение из пункта, пожалуйста, или пришлите фото в чат
```

### Перенастройки приставки
```txt
Потребуется перенастройка приставки:
1. Нажмите на пульте "Меню"
2. Откройте раздел "Настройки"
3. Нажмите на пункт "Сброс к заводским установкам" - "ОК"

После перезагрузки пройдите по инструкциям на экране ТВ
```

### Проверка типа получения сигнала

HLS или IPTV

```txt
Сейчас для установки типа сигнала:
1. Нажмите на пульте "Меню"
2. Откройте раздел "Настройки - "Тип получения сигнала"
3. Выберите "HLS" и подтвердите выбор

Напишите как сделаете
```

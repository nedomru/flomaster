## Развертывание проекта

Для развертывания проекта:
1. Клонируйте репозиторий
```bash
git clone https://github.com/AuthFailed/chat-flomaster.git
```
2. Установите зависимости
```bash
npm install
```
3. Запустите dev сервер Astro
```bash
npm run dev
```

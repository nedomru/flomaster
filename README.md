## Развертывание проекта

Для развертывания проекта:
1. Клонируйте репозиторий
```bash
git clone https://devops.chrsnv.ru/nedomru/flomaster.git
```
2. Установите зависимости
```bash
npm install
```
3. Запустите dev сервер Astro
```bash
npm run dev
```
